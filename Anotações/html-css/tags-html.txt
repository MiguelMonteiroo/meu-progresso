tags
<!DOCTYPE html> -> indica que é um documento em HTML5
<head> -> configurações do site
<body> -> conteúdo do site
<title> título do site
<h1> - <h6> -> títulos 
<p> -> parágrafos
<br> -> quebra de linha
<hr> -> linha horizontal
<img src="" alt=""> -> imagem
<picture> -> imagens flexíveis 
source media="" scrset="" type"" -> imagem alternativa
<link rel="" href="" type="image/x-icon"> -> favicon
<strong> -> negrito
<em> -> itálico
<del> -> texto deletado
<ins> texto inserido
<sup> -> texto sobrescrito
<sub> -> texto subscrito
<code> -> código-fonte
<pre> -> pré-formatação
<q> -> citação
<blockquote> -> citação com quebra de linha e deslocamento
<abbr> -> significado de abreviações
<bdo> -> inverter texto
<ol> -> lista ordernada
<ul> -> lista não ordenada
<li> -> item da lista
<dl> -> lista de descrição
<dt> -> termo
<dd> -> descrição do termo
<a> -> âncora
<a href="" rel="external/next/prev"> -> links externos/internos
<a href="" download=" type="" > link para download
